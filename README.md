# mnist-digit-classifier
A deep learning model that classifies handwritten digits (0–9) using the MNIST dataset, built with TensorFlow and Keras.
---
Overview

This model was trained on the MNIST dataset to recognize handwritten numbers from grayscale images of size 28×28 pixels.
---
The project achieved:

Training Accuracy: ~97%
Validation Accuracy: ~97%
Test Accuracy: ~97.6%
---
Technologies Used

Python
TensorFlow / Keras
NumPy
Matplotlib
---
Features

Train a neural network for digit recognition
Predict handwritten digits
Visualize prediction results
Evaluate model performance
---
Dataset

MNIST Handwritten Digits Dataset
70,000 grayscale images
10 classes (0–9)
